一文了解PID控制无刷电机

一、什么是PID？

PID的应用领域很广，本文简单介绍在无刷电机控制方面的应用。

首先要了解PID是什么，从名字上来看，PID的全称是：

比例(proportion)-积分(integral)-微分(derivative)控制器。

PID一般又分为：位置式PID，增量式PID

二、理解

首先我们通过比例(proportion)-积分(integral)-微分(derivative)，以及电机的例子三部分来理解PID。

1、电机模型建立

首先我们对电机的角度做PID控制。

设PID三个参数为Kp，Ki，Kd。

我们设电机返回的角度，也就是真实角度为：actual_angle，我们想要电机到达的目标角度为：target_angle，误差=真实-目标。假设电流正，电机转动方向为正方向。

2、PID理解与实现

如果我们单独使用比例(proportion)控制，假如actual_angle没有到达target_angle，那我就继续给电流，超过了，我就给负电流。那电流给什么值呢，如果是一个固定值，那么误差大和误差小，电机都以一样的速度转动，假如误差100度和1度，都用一个电流驱动，可能导致的问题就是，这个电流转很久到100度，或者会转的一下超出1度。所以应该误差大电流大，误差小电流小。

那么电流也就是最后的输出output = 一个系数 x 误差。乘以系数可以控制转速。

就称这个系数为比例系数，即Kp。
